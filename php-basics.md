# PHP Basics

## Usage
* Basic Method
```php
<? echo 'Hello World'; ?>
```
* XML Method
```php
<?php echo 'Hello World'; ?>
```
* Script Method
```php
<script language="php">
    echo 'Hello World';
</script>
```
* ASP Method
* php.ini file should be updated(asp_tags = ON) 
```php
<% echo 'Hello World' %>
```
